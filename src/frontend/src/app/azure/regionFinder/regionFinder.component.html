Cloud Region Finder

Cloud Region Finder enables you to quickly lookup cloud and region information of application deployment, try it by entering url or ip address now! Currently Azure, AWS, AliCloud are supported.

Restful API

Restful api is available via http://www.azurespeed.com/api/region, it returns datacenter region name. If the resolved address can not be found in azure datacenter ip ranges, returns region not found.

Examples:
  • https://azurespeed-api-wus.azurewebsites.net/api/region?ipOrUrl=65.52.170.210
  • https://azurespeed-api-wus.azurewebsites.net/api/region?ipOrUrl=www.azurespeed.com
