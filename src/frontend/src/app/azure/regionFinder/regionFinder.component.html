Restful API

Restful api is available via https://api.azurespeed.com/api/region, this api returns returns resolved Azure datacenter region information. If the resolved ip address can not be found in azure datacenter ip ranges, returns region not found.

Examples:

  • https://api.azurespeed.com/api/region?ipOrUrl=104.45.231.79
  • https://api.azurespeed.com/api/region?ipOrUrl=www.azurespeed.com
